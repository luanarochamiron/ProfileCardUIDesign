# Neumorphism Profile Card UI Design - CodingNepal

Um componente de cartão de perfil profissional (Profile Card) moderno, elegante e totalmente responsivo que adota a estética do Neumorfismo (Neumorphism / Soft UI). Desenvolvido exclusivamente com HTML5 e CSS3, o projeto utiliza o jogo de múltiplas sombras opostas para esculpir elementos táteis em relevo diretamente sobre a superfície da interface.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica do card, blocos textuais, ícones de controle, botões e dados de engajamento.
* CSS3: Estilização completa, uso de formas circulares, paleta monocromática suave e gerenciamento avançado de sombras projetadas.
* Flexbox: Distribuição espacial interna dos elementos e centralização absoluta do componente na viewport.

## Funcionalidades e Técnicas Aplicadas

* Estética Soft UI de Alta Definição: Aplicação precisa da propriedade `box-shadow` combinando luzes (brancas) e sombras sutis (cinzas) para criar o efeito de relevo extrudado no card, botões e moldura do avatar.
* Moldura de Avatar Neumórfica: Foto de perfil estruturada em formato perfeitamente circular, emoldurada por um anel esculpido em relevo côncavo/convexo.
* Botões de Ação Interativos: Botões "Message" e "Subscribe" estilizados como blocos táteis que reagem visualmente a interações do usuário.
* Redes Sociais e Dados de Engajamento:
  * Barra de ícones sociais circulares com cores oficiais discretas (Facebook, Twitter, Instagram e YouTube).
  * Contador inferior de métricas (Likes, Comments, Shares) distribuído de forma simétrica com divisores verticais sutis.
* Controles de Topo: Inclusão de botões de navegação rápida (seta de retorno e menu de opções de três pontos) integrados perfeitamente à cabeceira do card.
